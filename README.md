# TapSwap - مدیریت هوشمند کدهای تخفیف

TapSwap 是一个专门为波斯语用户设计的优惠码管理平台。它能自动收集、更新和管理来自各种来源的优惠码，让用户轻松找到并使用最新的优惠信息。

## 🌟 主要特点

- 🔄 **自动更新**: 每5分钟自动刷新优惠码
- 📅 **有效期管理**: 清晰显示优惠码的有效期和状态
- ✅ **智能验证**: 自动验证优惠码的有效性
- 🔔 **实时通知**: 新优惠码上线时获得通知
- 🌐 **波斯语支持**: 完全本地化的用户界面
- 💾 **本地存储**: 自动保存历史优惠码

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/yandong2023/tapswap.git

# 进入项目目录
cd tapswap

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 [http://tapswapcode.net](http://tapswapcode.net) 查看应用。

## 🛠️ 技术栈

- **框架**: Next.js 13
- **语言**: TypeScript
- **UI库**: Chakra UI
- **状态管理**: React Hooks
- **数据存储**: localStorage
- **样式**: Emotion
